# Deep-Learning-Style-Transfer-Project
## The project is divided into three main components:

1. Art Classification: Training and comparing VGG-19 and AlexNet architectures to distinguish authentic Van Gogh paintings from other Post-Impressionist works.

2. Neural Style Transfer (NST): Implementing a generative model to apply Van Gogh’s iconic brushwork and color palettes to new content images.

3. Cross-Model Evaluation: Using the trained "Art Expert" classifier to evaluate the structural integrity and stylistic success of the generated artwork.

## Key Features
* Architectural Comparison: In-depth analysis of how network depth (VGG-19) vs. computational efficiency (AlexNet) impacts the ability to separate semantic content from artistic texture.

* Performance Metrics: Evaluation of model sensitivity, including the identification of "circular validation traps" where shallower networks struggle with complex shape hierarchies.
